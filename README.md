
# ডিজিটাল টিচার - আটুলিয়া সোহরাবিয়া দাখিল মাদ্রাসা

এই প্রোজেক্টটি সরাসরি গিটহাব থেকে নেটলিফাইতে চালানোর জন্য কনফিগার করা হয়েছে।

### গিটহাব থেকে নেটলিফাই সেটআপ করার নিয়ম:
১. গিটহাবে ফাইলগুলো পুশ করুন।
২. নেটলিফাইতে **Import from GitHub** সিলেক্ট করুন।
৩. **Build Settings** এ:
   - **Build Command:** `npm run build` অথবা খালি রাখুন।
   - **Publish Directory:** `.` (একটি ডট দিন)।
৪. **Environment Variables** এ গিয়ে `API_KEY` নাম দিয়ে আপনার Gemini API Key টি দিন।

**নোট:** `netlify.toml` ফাইলটি অটোমেটিক বিল্ড এরর সমাধান করবে।
